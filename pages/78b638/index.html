# 略读

符合以下规则的句子直接忽略。

  1. XXXX。在此基础上，XXXX
  2. XXXX。换言之 XXXX
  3. XXXX，比如 XXXX
  4. 近年来 XXXX

# 中心理解题

选项会这么说：以下选项意在说明（）

“意在说明”、“意在强调”、“理解最准确的是”、“概括最恰当的是”……

# 标志词

虽然……但是……；尽管……可是……；不过；然而；却；其实/事实上/实际上

# 错误选项特征

  1. 提及转折前的内容
  2. 围绕解释说明的内容
  3. 无中生有

# 2. 因果关系

# 标志词

因为……所以……；由于……因此……

导致、致使、使得、造成

注意： 可见，看来 也是

# 思维导图

结论句出现在文段开头、中间，之后仍有其他语句

  1. 之后的语句是进一步解释说明，此时中心句仍为结论句

    公司福利待遇很好，所以我胖了。比如去年的衣服今年穿不上了。

  2. 之后又出现并列、因果、转折，需结合多种关联词共同分析

    因为考试前一天晚上没有休息好，所以考试没有发挥好。但是，我上岸了~

# 3. 必要条件关系

# 典型格式

只有 …… 才 …… , 后推前

理论要点： 必要条件才是重点！！ 必要条件即“只有”和“才”之间的部分

充分条件： 只要 A 就 B ，A => B

# 对策标志词及行文脉络

  1. 应该、应当、必须、需要、应、须、要+做法
  2. 通过/采耶……的手段/途径/措施/方式/方法/渠道，才能……
  3. 前提、基础、保障
  4. 负有……的义务/……的必由之路/……的法门之一/要领在于/势在必行

第4点常考！！！

# 反面论证的用法

典型格式：如果、倘若、一旦…… + 不好的结果

例：大量吃蒜不好……

A. 吃蒜要适度 B. 大量吃蒜不好

此时应该首选A。A是对策，有针对性的情况下， A 的优先级大于 B

# 4.并列关系

# 理论要点

全面概括

文段特征：

  1. 包含并列关联词，如 此外、另外、同时、“；”
  2. 句式相同或相近
  3. 按照时间顺序展开 (夏朝xxxx。明代中期xxxx。新中国以来xxxx)

错误选项特征：表述片面

巧用反向排除法

表述片面/主题词错误的直接视作干扰项

# 行文脉络

  1. 总-分
  2. 分-总
  3. 总-分-总
  4. 分-总-分
  5. 分-分

【理论要点】把握中心句及分述句的特点

【中心句特征】

形式：重点词提示

内容：观点（结论、对策、评价)

【分述句特征】

  1. 举例子：“比如”、“例如”、“……就是例证”等
  2. 数据资料（解释说明）
  3. 正反论证
  4. 并列分述：① 观点+个人；企业；国家；社会 ② 观点+举例+正反论证+原因

# 分总

# 标志词

对此、有鉴于此、尽管如此、在这个意义上、从这个角度来说

总之、换言之、简而言之、换句话说……

# 思维导图

尾句内容迷你眼，并非全部是重点

非重点处设陷阱，无中生有需警惕

# 做题技巧

【读题】：

  1. 在文段开头出现下定义、背景等内容不重要，直接略读
  2. 观点之后会进行论证，论证时会出现举例子、解释原因、调查报告/数据资料、正反论证，遇到时可略读。

【快速排除选项技巧】：

正确答案是对中心句的同义替换。

干扰选项常见错误为主题词错误，一定要抓准文段的主题词（如扩大/缩小/两个主题词表述片面）。

  1. 非重点表述：转折前、结论前、原因解释、原因、表述片面。
  2. 与作者的感情倾向相悖的需要排除。
  3. 表述绝对的需要排除，正确选项一般表述温和。
  4. 无中生有不能选。
  5. 不明确的选项不能选，一定要优选正确的，需要对比文段，对比择优。

# 细节判断

选项会这么说：

“以下对文段理解正确/不正确的是”

“符合/不符合这段话意思的是”

“从文段中可以得知/推出的是”

到底是选对的还是选错的 千万别看错了！！！

【做题顺序建议】

1.从文段特征看:

文段晦涩难懂:建议先看选项，再回到文段做比对

文段通俗易懂:可以先读文段，再看选项

2.从选项特征看:

选项特征明显，可以先从选项入手

【易定位特征】

  1. “三字一号”：数字、名字、字母、标点符号

  2. 核心名词：孤雌生殖或依赖雌性特有化学信号、使用超材料能够反弹雷达波

语句表达

语句表达共分为三块：语句排序、语句填空、接语选择

其中语句排序一般考2-3题；语句填空一般考2-3题，接语选择一般考1题

# 语句排序

【理论要点】

  1. 根据选项提示来确定首句

  2. 确定捆绑集团、确定顺序、确定尾句

# 确定首句

  1. 下定义：……就是/……是指/所谓……，指的是。

    不是看到下定义就一定做首句，而是对比内容来判断。

  2. 背景引入：随着、近年来、在……大背景/环境下。

    并不是看到“随着、近年来”就适合做首句，还是要对比择优。

  3. 非首句特征

    关联词后半部分

    指代词单独出现，指代不明确

    人称代词：他/她/它/他们

    指示代词：这/那/此

# 指代词捆绑

这、那、他、该、其/这些、它们

# 关联词捆绑

虽然……但是……

但、同时

# 时间顺序

【注意】时间顺序。

  1. 具体年份：2021 年、2022 年。

  2. 朝代：唐、宋、元、明、清。言语题目可能涉及到朝代的常识。

  3. 表示时间的词语：过去（后来）、现在、将来。“后来”即已经发生的事情。

  4. 拓展：朝代歌。夏朝往后考查较多。

    三皇五帝始，尧舜禹相传。

    夏商与西周，东周分两段。

    春秋与战国，一统秦两汉。

    三分魏蜀吴，两晋前后延。

    南北朝并立，隋唐五代传。

    宋元明清后，王朝至此完。

    民国再共和，代代往下传。

  5. 按照时间顺序排序要注意话题一致。

# 确定尾句

  1. 结论：因此、所以、看来、于是。

  2. 对策：这、应该、需要。

  3. 并不是看到“因此”“需要”就要做尾句，要对比内容来判断。有的题目可能结论词出现在开头和中间，后面进行解释。也有可能对策在开头和中间，后面还有具体的展开论述。

# 素材积累

坚守人民情怀，首要之处是尊重人民主体地位。“天地之大，黎元为先。” 人民是我们党执政的最大底气，是决定党和国家前途命运的根本力量。在近百年的奋斗历程中，我们党始终把人民立场作为根本立场，把为人民谋幸福作为根本使命，把全心全意为人民服务作为根本宗旨，把尊重人民主体地位和首创精神作为方法原则。

坚守人民情怀，关键环节是密切联系群众。“得众则得国，失众则失国。” 中国共产党成长发展的历史表明，我们党来自人民，为人民而生，因人民而兴。始终与人民心心相印、与人民同甘共苦、与人民团结奋斗，是我们党的光荣传统。可以说，我们党的历史就是密切联系群众而夺取革命、建设和改革事业伟大胜利的历史。

——中国青年报《牢牢坚守人民情怀》

因此，要把政治智慧的增长、执政本领的增强、领导艺术的提升、思想境界的提高，深深地扎根于人民群众之中，不断从人民群众及其参与的各项实践中汲取营养和力量。正所谓，“知屋漏者在宇下，知政失者在草野。” 当前，密切联系群众最直接的要求就是推动广大党员干部走近群众，扑下身子搞好调研，真正了解群众所思所想所盼，“练就一双火眼金睛”，从群众关注的烦心事、紧要事和身边事等细微之处入手，及时发现问题、精准研判问题、有效解决问题。

坚守人民情怀，目标指向是回应群众关切。“凡治国之道，必先富民。” 纵观古今中外，任何一个国家，其社会稳定、政权稳固都根源于人民群众的关切点和利益诉求得到满足，人民福祉得到保障。必须精准回应人民群众的利益诉求，切实解决广大人民群众的关切点和现实利益诉求，让人民群众有更好的教育、更高水平的医疗服务、更舒适的居住条件、更有预期的收入、更全面的社会保障、更丰富的精神文化生活。

——中国青年报《牢牢坚守人民情怀》

【注意】四个选项来自于中国青年报《牢牢坚守人民情怀》，积累下来可以用在写作中。

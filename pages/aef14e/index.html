变化类--概括变化

# 一、概念辨析

变化：某种情况发生了改变，过去和现在不一样

# 二、题型判断

题干中出现明显的“变化”倾向。

【例】

  1. 根据“给定资料1”，概括改革开放以来，赵家村发生的主要变化。

  2. 根据资料 2、3、5，谈谈“新型职业农民”的“新”体现在哪些方面。

# 三、答案呈现

# （一）一般情况：从……到……/从……变为……

表述上通常强调变化的过程——过去VS现在

【例】“给定资料1”中，风林村在实施“村寨银行”项目后，发生了巨大变化。请你谈谈风林村有了哪些变化。

一、经济水平变化。由谋生手段单一，收入微薄，贫穷落后，变为收入来源多样化，收入增加，实现脱贫致富。

二、生产方式变化。由消耗资源的粗放式生产，分散式小农经济，变为产业多元化、规模化，发展集体经济，实现绿色可持续发展。

# （二）特殊情况：仅写变化后的内容

适用情况：

  1. 材料中只有对现在的描述，没有过去的

  2. 过去的关键词和现在的关键词重合，重复表达太啰嗦；

如：基础设施愈发完善；生活水平明显提高

【例】（2019 国考地市）根据“给定资料3”，请分析小吴发生了哪些变化，使得村民们对她的称呼从“那个大学生”变成了“小吴”。（10分）

一、思想观念认识上，从自视甚高到理解农民，融通群众；从固执己见到听取领导意见，服务人民。

二、工作作风态度上，从空谈务虚到注重实践务实，深入田间地头，与农民共同劳作，帮大家解决教育、医疗等生活问题。

三、工作能力方式上，从不懂沟通到深入交流，从不懂变通到全面考虑，原则、方法、人情并重。

四、工作成果上，从无所事事转变为发挥专业优势，创建杂粮优质品牌并打入市场，带领农民增收致富。

【例】（2019 国考地市）根据“给定资料1”，请概括小张家乡出现的新变化。（15 分）

要求：准确全面，条理清晰。不超过200 字。

一、生产方式变化。利用技术提升产品附加值；从个体变为园区化、农场化、规模化、现代化。

二、销售方式转变。借助快递电商平台，从线下转到线上，从本地变为全国。

三、生活水平提升。收入增加，低保户减少；消费力增强，网购增多；闲暇时间多，娱乐丰富。

四、公共设施完善。道路硬化，建排水沟、垃圾箱，卫生条件改善，装修房屋，环境美化。

五、产业多元化。还乡人员增多，外来人口投资增加，加工工业、服务业发展迅速。

# 四、答题技巧

# （一）提示词

  1. 时间：之前是……，这几年是……；以前……，最近……；过去……现在……

  2. 对比：不再是，变得，更加；提高，增加，普及，改善，改进，革新，改良，丰富……

# （二）主题

定位主题，重点关注与主题相关的描述性信息。

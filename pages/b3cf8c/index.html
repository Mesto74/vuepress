基础知识

# 一、申论考试大纲

  1. 阅读理解能力——全面把握 给定资料 的相关内容，准确理解给定资料的含义，准确提炼 事实 所包含的观点，并揭示所反映的本质问题。

  2. 综合分析能力——对 给定资料 的全部或部分的内容、观点或问题进行 分析和归纳 ，多角度地思考资料内容，作出合理的 推断或评价 。

  3. 提出和解决问题能力—— 准确理解把握 给定资料所反映的 问题 ，提出 解决问题 的措施或办法。

  4. 文字表达能力——熟练使用指定的 语种 ，运用 说明、陈述、议论 等方式， 准确规范、简明畅达 地表述思想观点。

# 总结

  1. 测查基本能力——可通过练习提升。

  2. 客观性的考试——“材料为王”

  3. 部分答案要点需要通过归纳概括或分析推导资料得出。

# 二、答题要素与基础题型

# 单一题

题型判断：要素指向明确且数量较少；问题、原因、影响（意义、危害）、对策等。

赋分方式：按点（词）赋分。

【例 1】根据“给定资料2”，概括目前我国个人征信业务试点机构存在的主要问题。

要求：（1）内容全面，语言准确，条理清晰；（2）不超过 200 字。

【例 2】请根据“给定资料2”概括信用体系建设对当前社会具有的积极影响。

要求:（1）准确全面；（2）分条概括；（3）不超过 150 字。

# 综合题

题型判断：要素指向模糊且多元；问题、原因、影响（意义、危害）、对策等。

赋分方式：按点（词）赋分 + 层次赋分

【例 3】谈谈“预先失败”这一概念在“给定资料 4”中的含义。（10 分）

要求：全面、准确，不超过200 字。

【例 4】给定资料3 中提到：“尽管有些磕磕绊绊，但日子越过越有奔头”请根据给定资料3，谈谈你的理解。(20 分)

要求：（1）理解全面，分析透彻，层次清晰；（2）不超过 300 字。

# 公文题

题型判断：量词+文种，要素指向模糊、种类较多

赋分方式：

  1. 格式赋分 （各种公文的格式规范）；
  2. 内容赋分 ：关键词 + 层次分（分点，前后有层次）；
  3. 语言赋分 （加分项，要求辞藻，文段出彩）

【例 5】根据“给定资料6”，假定你是某高校团委负责人，请给学校党委写一份“加大信用教育力度”的建议书。（25 分）

要求:（1）紧扣资料，针对性强；（2）层次分明，逻辑性强；（3）格式规范，文体性强；（4）500字左右。

【例 6】外地的相关人员来学习农村垃圾治理的经验，假如你是 H 村的工作人员，请根据“给定资料 3”，写一篇介绍农村垃圾处理经验的发言提纲。（25 分）

要求：条理清晰，内容具体，不超过500字。

# 文章写作题

题型判断：一篇文章；一篇议论文

赋分方式：①内容客观（基于材料）；②文种正确（议论文）；③语言表达通顺流畅；④出现材料外的内容等。

【例 7】“给定资料1”中说“夜色难免黑凉，前行必有曙光”，“给定资料2”中说“我们的改革之所以成功，离不开务实的精神和理性的思考”。请深入理解这两句话的含义，参考给定资料，联系实际，自拟题目，写一篇文章。（35 分）

要求：（1）观点明确，见解深刻；（2）参考“给定资料”，但不拘泥于“给定资料”；（3）思路清晰，语言流畅；（4）字数1000～1200字。

【例 8】根据上述资料，以“梦想与定力”为主题，联系实际，自拟题目，写一篇议论文。（40分）

要求：（1）自选角度，立意明确；（2）思想清晰，语言流畅；（3）参考“给定资料”，但不拘泥于“给定资料”；（4）总字数1000 字左右。

# 三、答题步骤

# 审题方法

  1. 审作答范围：

    （1）根据、依据、结合、阅读给定资料X ：即范围在材料 X。

    （2）材料 X 中提到了 ：即范围重点在材料X，但要兼顾全部材料。

  2. 审作答主题与结构：

    主题：围绕 X 作答要素，X 即为主题。

    结构：答案的呈现形态

     【例 1】根据“给定资料1”，概括s 市在乡风文明建设方面的举措。

 要求：全面、准确，不超过150 字。

 【例 2】谈谈“预先失败”这一概念在“给定资料 4”中的含义。

 要求：全面、准确，不超过200 字。

  3. 审答题要求：

    准确：一是要点来自材料；二是最好使用材料原词原句，或在原词原句的基础上归纳、概括。

    全面：确保关键词无遗漏。

    简明：表述言简意赅，尽量突出关键词，去掉修饰语，避免同义反复。

    有条理：分条撰写，要标注序号。

    层次分明：建议分类。

# 答案组织

  1. 加序号，分条书写：

    常见的方法是“1.2.3” “①②③” 或 “一、二、三”。

  2. 分类整理：

    （1）分类标志词：分别、种种、归纳、层次分明、分类合理……

    （2）分类逻辑：题干逻辑、材料逻辑、要素逻辑。

问题类--基础理论

# 常见问法

关键在于抓主题词 + 负面情况：

（2019 黑龙江公检法）根据“给定资料2”，概括目前我国个人征信业务试点机构存在的主要问题。

（2014 国考副省）“给定资料2”揭示了当前社会心理方面存在的若干“缺失”，请对此予以归纳概括。

（2019 吉林甲）根据给定资料2、3，概括我国民间智库发展过程中面临的困难。

# 答题技巧

# （一）直接摘抄

寻找与主题有关的负面的标志性词汇并摘抄相关内容，如：

不科学、不合理、不到位、不均衡、不完善、不健全、不足；缺乏、缺少、流失；

少、低、差、弱；陈旧、单一、阻碍、桎梏等。

【例】根据给定资料，分析C市目前幼儿教育发展的主要问题。

据悉 C 市有相当一部分民办幼儿园占地面积、房屋建筑面积达不到要求，办学规模较小，设备设施简陋，办园条件差，教师待遇偏低。

直接摘抄问题：民办幼儿园占地面积、建筑面积不达标，办学规模小，设备设施简陋，办园条件差，师资待遇低。

特别强调：A.抄完整结构。B.基于主题之下的负面信息，要关注主题。

# （二）归纳概括

对表述不规范、冗长的负面资料进行归纳概括，如对数据型资料、案例型资料等进行精准归纳概括。

# （1）数据型资料

看数据型资料前后有无总结句，有总结句，优先抄总结句；

若没有总结句，要通过分析，用“多少、大小、高低、快慢、增减、比重”等词概括数字背后反映的趋势。

【例】针对博物馆、美术馆、科技馆、纪念馆和陈列馆的调查数据显示，24.4%的受访者一年中从未去过这 5 类场所，61.6%的人平均每年去1到5次。而市民对去文化馆的次数更低，93.7%的人平均每年到馆 5 次以下，其中，58.6%的人没去过文化馆，35.1%的人每年到馆次数为1 到5 次。

归纳概括问题：市民参观文化场馆的次数少/积极性差/意愿低

注意：归纳概括后表意的精确性，不可改变资料本意；储备规范词

# （2）案例型资料

对表述不规范的负面资料进行归纳概括，如数据型资料、案例型资料等。

【例】概括 C 市目前幼儿教育发展的主要问题。王女士是该市的一位普通家长，她的孩子5岁，在当地一所民办幼儿园就读。幼儿园老师会号召班里的孩子们，孤立疏远那些“上课调皮”的孩子，不理他们，不和他们玩。

归纳概括问题：教师师德差/师德缺失/道德素养低下。

# 总结

  1. 归纳概括后表意的精确性，不可改变资料本意；

  2. 归纳概括需要储备一定数量的规范词

# （三）分析推导

对建议型表达或对比型表达进行分析推导。

# （1）对建议型表达进行分析推导

【例】应对精神疾病等心理健康问题，我们国家应该建立起完善的心理疏导机制，还要把心理疾病当成生理疾病来同等对待，同时严格执行心理咨询机构的准入标准。

分析推导问题：心理疏导机制不完善，对心理疾病不重视，心理咨询机构的准入标准执行不严格。

# （2）对对比型表达进行分析推导

A.横向对比：不同主体之间的对比

B.纵向对比：同一主体不同时期的对比

【例】分析我国控制过度包装工作存在的问题。

每年全国城市固体废弃物中，包装物占30%，每年废弃价值达 4000 亿元，而处理这些“美丽垃圾”的费用更是惊人。而在这方面，国外有很多成功的经验可循。比如英国几年前也曾在全国上下掀起了一场关于过度包装问题的热烈讨论；另外，以关心环保问题闻名的《独立报》推出了一项“反包装浪费”活动，要求超市和零售商为各种商品“脱掉多余的外衣”。

分析推导问题：未形成理性消费的社会氛围，缺乏正确的媒体舆论导向。
